# Настройка Google Apps Script

## Проблема с камерой

Камера **НЕ работает** внутри Google Apps Script из-за ограничений безопасности (Sandbox iframe).

## Решение: Используйте локальную версию

Создайте **локальную версию** с подключением к Google Таблице:

### Шаг 1: Настройте Google Apps Script (серверная часть)

1. Откройте https://docs.google.com/spreadsheets/d/1Bp6c3Ws8rTvj2TW7YxiAreyyZ8WSdX9cXj3M9C48AOM/edit
2. **Расширения** → **Apps Script**
3. Создайте 2 файла:

#### Файл: Code.gs
Скопируйте содержимое из файла `Code.gs`

#### Файл: index.html
Скопируйте содержимое из файла `index.html` (из папки phone-scan в проекте)

4. Нажмите **Развернуть** → **Новое развёртывание**
5. Выберите **Веб-приложение**
6. Настройте:
   - Описание: `Версия 1`
   - Выполнять как: `Я`
   - Кому доступно: `Все`
7. Нажмите **Развернуть**
8. **Скопируйте URL** (например: `https://script.google.com/macros/s/ABC123XYZ/dev`)

### Шаг 2: Настройте локальный HTML (клиентская часть)

1. Откройте файл [`index_standalone.html`](phone-scan/index_standalone.html)
2. Найдите строку (примерно 207):
   ```javascript
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec';
   ```
3. Замените `YOUR_SCRIPT_ID` на ваш URL из шага 1

### Шаг 3: Запустите

Откройте `index_standalone.html` в браузере:
- Можно просто перетащить файл в браузер
- Или запустить через `node serve.js`

**Камера будет работать!** ✅

---

## Альтернатива (если не нужна камера)

Если камера не нужна, можно использовать версию полностью в Google:
- index.html + Code.gs + css.html в Apps Script
- Но камера не будет работать
