# Сканер рулонов ткани

Мобильное веб-приложение для инвентаризации рулонов ткани с QR-кодами.

## Ссылки

- **GitHub:** https://github.com/artteth/phone-scan
- **Google Sheets API:** https://script.google.com/macros/s/AKfycbxrD_B4A37h6QlyXgn4TX_nQYHgwGfkTAj4qVBD5-g_OM9-nMJjU1zhvIcMX4yztk36/exec

## Технологии

- HTML5 + JavaScript
- html5-qrcode — сканирование QR-кодов через камеру
- jsQR — альтернативный сканер (ZBar)
- Scanbot SDK — третий сканер (60 сек бесплатно)
- Google Sheets API — синхронизация данных

## Использование

1. Открыть приложение
2. Нажать "Сканер QR-код"
3. Разрешить доступ к камере
4. Навести на QR-код

### Поиск заказов

На странице "Все заказы" есть поле поиска. При нажатии на него появляется выпадающий список со всеми существующими заказами из Google Таблицы. Можно:

- Нажать на заказ в списке для быстрого выбора
- Ввести номер заказа вручную для фильтрации списка
- Список автоматически обновляется при синхронизации с Google Таблицей

### Формат QR-кодов

QR-код должен содержать номер заказа и номер рулона через нижнее подчёркивание:
- `2020_1` — заказ 2020, рулон 1
- `2258_5` — заказ 2258, рулон 5

## Настройки

Нажмите ⚙️ в правом верхнем углу для настройки частоты кадров сканера (FPS).
