# Сканер рулонов ткани

Веб-приложение для инвентаризации рулонов ткани с интеграцией в Google Таблицы.

## 🚀 Быстрый старт

### Шаг 1: Настройте Google Apps Script

1. Откройте вашу Google Таблицу: https://docs.google.com/spreadsheets/d/1Bp6c3Ws8rTvj2TW7YxiAreyyZ8WSdX9cXj3M9C48AOM/edit
2. Перейдите в **Расширения** → **Apps Script**
3. **Важно!** Создайте файл **Code.gs**:
   - Нажмите "+" (добавить файл) → **gs файл**
   - Назовите **Code.gs**
   - Скопируйте код из файла [Code.gs](https://github.com/artteth/phone-scan/blob/main/Code.gs)
4. Нажмите **Сохранить** (иконка 💾)
5. Нажмите **Развернуть** → **Новое развёртывание**
6. Выберите тип **Веб-приложение**
7. Настройте:
   - Описание: `Версия 1`
   - Выполнять как: **Я**
   - Кому доступно: **Все**
8. Нажмите **Развернуть**
9. **Скопируйте URL** веб-приложения

### Шаг 2: Обновите ссылку в приложении

1. Откройте файл [index.html](https://github.com/artteth/phone-scan/edit/main/index.html)
2. Найдите строку (~207):
   ```javascript
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/.../exec';
   ```
3. Замените URL на ваш из Google Apps Script
4. Сохраните и запушьте изменения

### Шаг 3: Откройте приложение

- **https://artteth.github.io/phone-scan/**

## 📱 Функции

- 📷 Сканирование QR-кодов
- 📝 Запись измерений (заводской/измеренный метраж, усадка)
- 📋 Управление заказами
- ☁️ Сохранение данных в Google Таблицу

## 📊 Структура таблицы

Скрипт автоматически создаст лист "Inventory" с заголовками:

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| ID заказа | Номер рулона | Заводской метраж | Измеренный метраж | Усадка (%) | Статус | Дата обновления |

## ⚠️ Ошибка подключения

Если видите "Не удалось подключиться к Google Таблице":

1. Проверьте что файл **Code.gs** существует в Apps Script
2. Убедитесь что нажали "Развернуть" → "Новое развёртывание"
3. Проверьте что URL в index.html совпадает с URL из Apps Script
