# Сканер рулонов ткани

Веб-приложение для инвентаризации рулонов ткани с интеграцией в Google Таблицы.

## 🚀 Быстрый старт

### 1. Настройте Google Apps Script

1. Откройте вашу Google Таблицу
2. Перейдите в **Расширения** → **Apps Script**
3. Создайте файл **Code.gs** и вставьте код из файла [Code.gs](Code.gs)
4. Нажмите **Развернуть** → **Новое развёртывание**
5. Выберите тип **Веб-приложение**
6. Настройте:
   - Выполнять как: **Я**
   - Кому доступно: **Все**
7. Нажмите **Развернуть** и скопируйте URL

### 2. Подключите к приложению

1. Откройте файл [index.html](index.html)
2. Найдите строку (строка ~207):
   ```javascript
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec';
   ```
3. Замените `YOUR_SCRIPT_ID` на ваш URL из Google Apps Script

### 3. Откройте приложение

- Через GitHub Pages: https://artteth.github.io/phone-scan/
- Или локально: откройте index.html в браузере

## 📱 Функции

- 📷 Сканирование QR-кодов
- 📝 Запись измерений (заводской/измеренный метраж, усадка)
- 📋 Управление заказами
- ☁️ Сохранение данных в Google Таблицу
- 📴 Работа офлайн (с синхронизацией)

## 📊 Структура таблицы

| Столбец | Название |
|---------|----------|
| A | ID заказа |
| B | Номер рулона |
| C | Заводской метраж |
| D | Измеренный метраж |
| E | Усадка (%) |
| F | Статус |
| G | Дата обновления |

## 🔧 Технологии

- HTML5, CSS3, Vanilla JavaScript
- Google Apps Script (backend)
- html5-qrcode (сканер)
- QRCode.js (генератор)
